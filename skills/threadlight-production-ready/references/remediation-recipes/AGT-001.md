## Target file
`src/app.py` (or `src/main.py` / `src/agent/main.py` / `src/index.ts` — the main agent entry-point).

## Edit type
`insert` — add the AGT import at the top of the file if missing.

## Edit recipe
1. Locate the main agent container entry-point (typically `src/app.py` for Python, `src/index.ts` for TypeScript).
2. At the top of the file, after any system imports, add:

   **Python:**
   ```python
   from agt import apply_governance
   ```

   **TypeScript:**
   ```typescript
   import { applyGovernance } from '@foundry/agt';
   ```

3. Wire the middleware before any agent initialization:

   **Python:**
   ```python
   # After imports, before agent startup
   governance = apply_governance(config_file='policy.yaml')
   ```

   **TypeScript:**
   ```typescript
   // After imports, before agent startup
   const governance = applyGovernance({ configFile: 'policy.yaml' });
   ```

4. If the agent uses a framework (FastAPI, Express, etc.), ensure the governance wrapper is applied to the agent handler, not bypassed by direct LLM calls.
