## Target file
`policy.yaml` (agent governance policy file).

## Edit type
`replace`

## Edit recipe
1. Locate `policy.yaml` and find the `responsible_ai` → `prompt_shields` or `jailbreak_protection` section (created or updated in RAI-002).
2. Ensure the following fields are explicitly set to `enabled`:

   ```yaml
   responsible_ai:
     prompt_shields:
       jailbreak: enabled
       indirect_attack: enabled
   ```

3. If the fields are missing, add them. If they are set to `disabled` or `false`, replace with `enabled` or `true`.
4. Explanation of shields:
   - `jailbreak`: Detects and rejects prompts attempting to override system instructions.
   - `indirect_attack`: Detects and rejects multi-hop prompts designed to extract harmful information indirectly.
