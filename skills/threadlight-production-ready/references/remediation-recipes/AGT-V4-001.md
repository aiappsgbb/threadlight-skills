## Target file
`requirements.txt` (Python) or `package.json` (JavaScript/TypeScript); check both if the repo uses multiple runtime languages.

## Edit type
`replace` — update AGT dependency declaration to v4-specific distribution name.

## Edit recipe
1. Locate the AGT dependency in the applicable dependency file:
   - **Python:** Search in `requirements*.txt`, `pyproject.toml`, or `setup.py` for any entry matching `agent-governance-toolkit`, `foundry-agt`, or `agt`.
   - **JavaScript:** Search in `package.json` for entries under `dependencies` or `devDependencies` matching `@foundry/agt`, `agent-governance-toolkit`, or similar.

2. If found, replace the v3.7-style declaration with a v4-specific name:

   **Python (requirements.txt):**
   - Old: `agent-governance-toolkit==3.7.0` → New: `agent-governance-toolkit-core==4.1.0`
   - Or: `foundry-agt>=3.7` → New: `agent-governance-toolkit[full]==4.1.0`

   **Python (pyproject.toml):**
   ```toml
   [project]
   dependencies = [
       "agent-governance-toolkit-core==4.1.0",  # or -runtime, -sre, -cli, or [full]
   ]
   ```

   **JavaScript (package.json):**
   ```json
   {
     "dependencies": {
       "agent-governance-toolkit-core": "^4.1.0"
     }
   }
   ```

3. Valid v4 distribution names (one required):
   - `agent-governance-toolkit-core` — core governance engine
   - `agent-governance-toolkit-runtime` — runtime integration
   - `agent-governance-toolkit-sre` — SRE/observability extensions
   - `agent-governance-toolkit-cli` — CLI tooling
   - `agent-governance-toolkit[full]` — all distributions (umbrella)


Expected: at least one match with a v4-specific distribution name.
