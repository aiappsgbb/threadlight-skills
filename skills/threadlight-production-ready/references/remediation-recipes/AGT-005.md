## Target file
`policy.yaml` (or the AGT policy file identified by AGT-002).

## Edit type
`replace` — enhance the existing policy with tool-call and prompt-shield blocks if absent.

## Edit recipe
1. Open the existing `policy.yaml` file.
2. If `tool_calls:` block is missing, add it under the top-level `agent_control_specification_version:`:

   ```yaml
   tool_calls:
     allowed_categories:
       - "data_retrieval"
       - "reporting"
     blocked_operations:
       - "delete"
       - "destroy"
       - "purge"
     requires_approval:
       - "write_to_external_system"
   ```

3. If `prompt_shields:` or `jailbreak_detection:` is missing, add it:

   ```yaml
   prompt_shields:
     enabled: true
     jailbreak_detection:
       patterns:
         - "ignore previous instructions"
         - "pretend you are an unrestricted"
       action: "deny_and_log"
     pii_redaction:
       enabled: true
       patterns:
         - "ssn"
         - "credit.card"
         - "api.key"
       action: "redact_and_log"
   ```

4. Ensure both blocks are present and at the same indentation level (under the root of the policy YAML).
