## Target file
`infra/identity.bicep` (create if missing, or insert into `infra/main.bicep`). The check looks for `Microsoft.ManagedIdentity/userAssignedIdentities` in compiled ARM.

## Edit type
`insert`

## Edit recipe
1. Create `infra/identity.bicep`:

   ```bicep
   param location string
   param environment string

   resource workloadIdentity 'Microsoft.ManagedIdentity/userAssignedIdentities@2023-01-31' = {
     name: 'mi-agent-${environment}-${uniqueString(resourceGroup().id)}'
     location: location
   }

   output identityId string = workloadIdentity.id
   output identityPrincipalId string = workloadIdentity.properties.principalId
   output identityClientId string = workloadIdentity.properties.clientId
   ```

2. Reference it in `infra/main.bicep`:

   ```bicep
   module identityResources 'identity.bicep' = {
     name: 'identityDeployment'
     params: {
       location: location
       environment: environment
     }
   }

   // Export for use by Container App / Functions / other workloads
   output workloadIdentityId string = identityResources.outputs.identityId
   output workloadIdentityPrincipalId string = identityResources.outputs.identityPrincipalId
   ```

3. Assign this identity to your workload (e.g., Container App):

   ```bicep
   resource containerApp 'Microsoft.App/containerApps@2023-05-01' = {
     name: 'agent-app'
     location: location
     identity: {
       type: 'UserAssigned'
       userAssignedIdentities: {
         '${identityResources.outputs.identityId}': {}
       }
     }
     properties: {
       // ... other properties
     }
   }
   ```

4. Grant RBAC roles to the identity (see IAM-003 for scoping examples).
