## Target file
`infra/observability/loganalytics.bicep` (create if missing; search for `Microsoft.OperationalInsights/workspaces` if path differs).

## Edit type
`insert`

## Edit recipe
1. Create or open `infra/observability/loganalytics.bicep`.
2. If the file is empty or does not exist, insert the following Log Analytics workspace resource:

```bicep
param location string
param workspaceName string
param retentionInDays int = 90
param tags object = {}

resource logAnalyticsWorkspace 'Microsoft.OperationalInsights/workspaces@2021-12-01-preview' = {
  name: workspaceName
  location: location
  properties: {
    retentionInDays: retentionInDays
    publicNetworkAccessForIngestion: 'Enabled'
    publicNetworkAccessForQuery: 'Enabled'
    sku: {
      name: 'PerGB2018'
    }
  }
  tags: tags
}

output workspaceResourceId string = logAnalyticsWorkspace.id
output workspaceId string = logAnalyticsWorkspace.properties.customerId
output workspaceKey string = listKeys(logAnalyticsWorkspace.id, '2021-12-01-preview').primarySharedKey
```

3. Ensure this workspace is referenced from the main Bicep template and that its resource ID is passed to the App Insights resource (OBS-001).
4. Set `retentionInDays` to at least 90 for production posture.
