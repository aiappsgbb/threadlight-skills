The five-stage funnel

From a paragraph to a governed agent — in five named stages.

Every Threadlight engagement walks the same five stages. Each stage has its own skills, its own artefacts, and its own exit criteria.

01 · Conversation Describe the process Plain-language brief from you or a customer SME. No template, no questionnaire. The skills chain

You prompt. The coding agent picks up the right skills.

Skills are composable building blocks — Markdown instructions that any coding agent (GitHub Copilot CLI, VS Code, Claude Code) reads and follows. You describe what you need in a prompt. The agent discovers the matching skill, runs it, and hands you the artefacts — all the way to a governed agent on your Azure with a paved path to production via AI Citadel (the governance hub) and the Agent Governance Toolkit.

What it looks like in the field

Six industries. One pattern.

The same five-stage funnel ships agents across regulated and unregulated industries. The skills don't change — the SPEC does. Below: one example process per industry we work in.

🏦

Financial services

Claims triage

Route inbound claims by complexity and regulator exposure. Spec, deploy, evaluate — in-session.

spec → deploy → safe-check 🏥

Healthcare

Prior-authorization assistant

Generates structured payor submissions from the clinical narrative — with HITL gates wired from day zero.

spec → HITL patterns → deploy 🏭

Manufacturing

Field-service knowledge agent

Technicians ask the agent. The agent grounds against equipment manuals + service history — with cost-projected SKUs picked before go-live.

spec → deploy → cost 🛍️

Retail

Merchandising co-pilot

Buyer asks for promo plans; agent drafts them grounded in last-quarter SKU performance and store-cluster lift.

spec → demo-data → deploy 🏛️

Public sector

Citizen-facing case status

Public-language status updates with case-worker hand-off built in. Governance and audit aren't advisory here — they're hard gates the chain wires before a single message ships.

spec → HITL → production-ready 📡

Telco

Network-incident triage

Cross-references SNMP, ticket history, and runbooks into a first-pass remediation — cost-projected against the production load profile.

spec → deploy → consumption-iq

All six follow the same five-stage funnel. The differences live in the SPEC, not the chain. Read the full industries chapter →
