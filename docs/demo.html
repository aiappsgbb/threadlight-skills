The sales kit — generated, not slideware

It also writes your pre-sales kit.

Once the deployment passes threadlight-safe-check, threadlight-consumption-iq reads the real provisioned estate and turns it into a phased Azure cost estimate and a shareable seller one-pager — every figure framed as a planning estimate, never a quote. It runs before the production-readiness scorecard.

safe-check ✓ post-deploy → consumption-iq sales kit → production-ready scorecard

  • ⌗
    Phased, not one scary number

    PoC → expansion (current) → business-wide, each an estimate at public list price — so the ramp is expected, not a step-change.

  • %
    EA framing built in

    A 15% EA multiplier is shown as an internal planning assumption beside list price — clearly labelled an estimate, never a contractual rate.

  • ❝
    A seller talk-track, written for you

    “Lead with Phase 1. Never quote Phase 3 alone. Qualify every EA figure.” The how-to-open-the-conversation script ships in the page.

  • ⚑
    Internal by design

    Stamped do not share with the customer. Estate-wide shared platform (Defender · Sentinel · DDoS) is amortised, not charged to one app.
